Question-3:What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of multiple event listeners on individual child elements. This is possible because of event bubbling, where events triggered on child elements propagate up to their ancestors.

Question-4:What is event bubbling in the DOM?

Event bubbling is a concept in the DOM (Document Object Model) event propagation mechanism, where an event that occurs on a child element first triggers the event handler on that element and then propagates (or "bubbles up") to its parent elements until it reaches the root (document)..
